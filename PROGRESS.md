## 项目文件结构

```
yigou-ide/
├── backend/
│   ├── core/
│   │   └── code_converter.go      # 中文代码转换器（100+函数映射）
│   ├── services/
│   │   ├── build_service.go       # 编译打包服务
│   │   ├── file_service.go        # 文件读写服务
│   │   └── project_service.go     # 项目管理服务（SQLite）
│   └── models/
│       └── project.go             # 项目数据模型
├── frontend/src/
│   ├── components/
│   │   ├── build/
│   │   │   └── BuildDialog.vue    # 构建配置对话框
│   │   ├── designer/
│   │   │   ├── ComponentLibrary.vue  # 组件库面板
│   │   │   ├── DesignCanvas.vue      # 设计画布
│   │   │   ├── PreviewDialog.vue     # 预览对话框
│   │   │   └── PropertyPanel.vue     # 属性面板
│   │   ├── editor/
│   │   │   └── BottomPanel.vue       # 底部面板（代码/控制台）
│   │   ├── layout/
│   │   │   └── TopMenuBar.vue        # 顶部菜单栏
│   │   └── project/
│   │       └── ProjectManager.vue    # 项目管理器
│   ├── stores/
│   │   └── theme.ts                  # 主题状态管理
│   ├── styles/
│   │   └── theme.css                 # 主题CSS变量
│   └── App.vue                       # 主应用组件
├── app.go                            # 应用核心（API暴露）
├── main.go                           # 入口文件
├── wails.json                        # Wails配置
├── go.mod                            # Go模块
└── PROGRESS.md                       # 进度文档
```
